# Documemt_AncientWarToolkit2.0 API 文档
## 概述

AncientWarToolkit2.0 API是[逆转命运工作室](http://www.rdcraft.cn/about/)自[《RD 古代战争：维京传奇》](https://b23.tv/vpyZTaj)版本后开发后续古代战争系列作品的基石，能让团队开发者通过更简单的方式实现复杂的游戏内容。
